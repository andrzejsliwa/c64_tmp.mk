# c64_tmp.mk

Example project for C64 Turbo Macro Pro (1.2)

## Usage

```bash
*[master][~/c64_tmp.mk]$ make help
Help:
  make start               build and start emulator (optionally with name of program)
  make debug               build and run in debugger (optionally with name of program)
  make clean               clean build directory
  make compile             compile ASSEMBLER source files (src/*.a)
  make convert_basic       convert BASIC source files (src/*.bas)
  make convert_asm         convert ASSEMBLER source files (build/*.a,*.i)
```
